
Build
-----
$ npm install -g cordova
$ npm install -g grunt-cli

$ cd locator

$ npm install
$ grunt 

$ cordova platform add ios
$ cordova platform add android

$ cordova prepare
$ cordova requirements

$ cordova build

$ cordova run android
